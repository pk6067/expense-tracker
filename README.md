A professional-grade, mobile-responsive expense tracker built with Vanilla JavaScript.

## ✨ Key Features
- **Dynamic Visualization**: Uses Chart.js to provide a real-time doughnut chart of spending by category.
- **Persistent Storage**: Leverages the Browser's LocalStorage API to save data without a database.
- **CRUD Functionality**: Create, Read, and Delete transactions with instant UI updates.
- **Mobile First**: Fully responsive CSS using Flexbox and CSS Variables.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Custom Variables), JavaScript (ES6+)
- **Charts**: Chart.js
- **Deployment**: GitHub Pages

## 📖 How to Run
1. Clone the repo: `git clone [repo-url]`
2. Open `index.html` in your browser.
